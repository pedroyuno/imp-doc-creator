üìù Row Descriptions

Row 1: Feature Column Names

Names for each provider + payment method combination

Examples: "REDE_CARD", "PAGARME_PIX", "[Provider]"
Row 2: Provider Names

Actual provider names for valid combinations

Valid: "REDE", "PAGARME" | Invalid: "#N/A", empty
Row 3: Payment Methods

Payment methods for each provider

Valid: "CARD", "PIX", "BOLETO" | Invalid: "#N/A", empty
Rows 4+: Feature Data

Integration features and their values

Feature names in column 1, values in corresponding provider columns
